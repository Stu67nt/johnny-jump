# Johnny Jump

This is a platformer game (technically) where you can jump around ant try to reach the top! (there are no prizes but aura of reaching the top)

This project is for Hack Club's #ResolutionGameDev event, where each week you work on a game dev project relating to a theme! This project is week 3 of Resolution, where I needed to create a simple platformer.

For this, I took the base concepts I learned and added upon it by modifying the movemeent script to implement double jump and also added onto the level. 

I found the assets from [this repo](https://github.com/LunaTMT/Doodle-Jump)
